Join the Breez SDK Beta Program

We're opening the beta program for our new Nodeless implementation of the Breez SDK, built on Spark's Bitcoin-native Layer 2, making it even easier to bring bitcoin into everyday apps and services.

Fully Bitcoin-native, Spark is a shared signing protocol enabling instant, low-fee, self-custodial transactions, with full support for Lightning and native stablecoins issued using its token standard (BTKN).

As part of the beta, you'll get early access to Breez's best-in-class Lightning SDK, including bindings for all major languages and frameworks, support for LNURL, Lightning addresses, BOLT11, BOLT12, BIP353, Nostr Wallet Connect, WebLN, and more features to come.

Note: Not all features will be available in the beta program
