Breez SDK Beta

Introducing Nodeless on Spark

We're excited to open the beta program for the new Nodeless implementation of the Breez SDK, built on Spark, making it even easier for developers to bring bitcoin into everyday apps and services.


Spark is a Bitcoin-native Layer 2 delivering a faster, cheaper way to build on bitcoin. It offers real-time, low-fee, self-custodial payments and support for Bitcoin-native assets like stablecoins.


As part of the beta, you'll get early access to Breez's best-in-class Lightning SDK, with bindings for all major languages and frameworks. You'll also have built-in support for LNURL, Lightning addresses, BOLT12, BIP353, Nostr Wallet Connect, WebLN, and more features to come.
